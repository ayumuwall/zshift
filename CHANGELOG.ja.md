# 変更履歴

[English](CHANGELOG.md)

このプロジェクトの主な変更点を記録します。

## [0.1.2] - 2026-03-27

### 修正
- Shift+←/→ のディレクトリナビゲーション修正: `_ZSHIFT_LIST_ALL_CMD` をインライン展開していたことによる awk スクリプトのクォート崩れを解消
- Shift+← の修正: `b` の値を壊れた action 構文ではなく `printf %s` 経由で渡すよう変更
- 複数選択の修正: zsh での改行分割を確実にするため `${(f)picks}` を使用
- Shift キーナビゲーションのバインディングをデバッグ・安定化

### 追加
- fzf ヘッダーの i18n 対応: `LANG`/`LC_ALL` のロケールに応じて英語・日本語を自動切替

### ドキュメント
- 英語 README を追加し、日本語訳へのリンクを設置
- README にスクリーンショットを追加し、Demo セクションを刷新
- README.ja.md の全見出しを日本語化
- 機能比較表から不正確な行を削除

### インストール
- Homebrew インストール手順に `source ~/.zshrc` の実行ステップを追加

## [0.1.1] - 2026-03-12

### 修正
- Homebrew Formula のインストールパス修正: `share/zshift` サブディレクトリを正しく作成するよう変更

## [0.1.0] - 2026-03-12

### 追加
- 初回リリース: zshift — zsh のための強化版 Ctrl+T
  - fzf を使ったパス認識型 Ctrl+T ファイル・ディレクトリピッカー
  - Shift+← / Shift+→ でピッカーを離れずにディレクトリを移動
  - `~` キーでホームディレクトリへ即移動
  - ディレクトリをシアン・ファイルをグレーでカラー表示
  - 複数選択対応
  - zoxide 連携 (`z` / `zi`)
  - Homebrew Formula による簡単インストール
